Princípios do Patrimonium DS 5 princípios que guiam decisões de design. Quando algo não estiver no sistema, decida com base aqui — não invente padrão novo se um princípio resolve.
01

Solidez antes de virtuosismo

A Patrimonium é uma marca financeira e contábil. Confiança vem antes de personalidade. O sistema favorece soluções discretas, profissionais e duradouras em vez de decisões expressivas que pareçam modinha. Quando estiver em dúvida entre o sóbrio e o brilhante, escolha o sóbrio.

02

O miolo é sagrado

O elemento capsular verde do logo (o miolo do P) é a identidade visual mais reconhecível da marca. Toda decisão de forma, cor e ritmo no DS conversa com essa identidade — direta ou indiretamente. Não há decisão neutra sobre o miolo.

03

Discreet, low-spread

Sombras, transições e efeitos no sistema são sutis, contidos e sem drama. Nada de brightness alto, blur exagerado, animações longas. Profissionalismo é evidenciado por contenção, não por exuberância.

04

Pill é motivo, não enfeite

O radius/pill (999px) é a forma extrema da identidade Patrimonium. Aparece pouco, significa muito. Reservado para elementos que carregam a marca diretamente: botão affirmative, dots, logo, status circular, switch track. Usar fora desses casos dilui o significado.

05

Branded House, não casa de marcas

As 8 sub-marcas compartilham o mesmo chassi visual. A diferenciação acontece pela logo (ícone + nome), nunca por cor extra. Toda sub-marca usa Patrimonium Blue + Green + construção outline. Inventar cor ou tipografia própria para uma sub-marca é proibido.

Adendo · Opacidade não é foundation — Opacidade nunca é usada para alterar a aparência de conteúdo significativo. Para atenuar texto, use o token de cor certo (--neutral-light). Para "desligar" um botão, use a variant disabled. Para fundos translúcidos funcionais, use os 3 tokens autorizados: --backdrop-modal, --focus-ring, --menu-hover-on-dark/light.
