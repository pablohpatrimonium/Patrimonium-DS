v1.0.0 maio 2026 Primeira versão pública do Patrimonium Design System.

Adicionado

  • Foundations — Colors (Brand + Apoio + Semantic com -bg), Typography (RNS Sanz, 10 estilos), Spacing (11 tokens), Radii (escala + exceção mini-controle), Shadows (3 níveis), Grid (5 breakpoints, mobile-first), Iconografia (Lucide, stroke 1.5px, 4 tamanhos), Acessibilidade (Contraste WCAG + 6 camadas estruturais).
  • Atoms — Button (5 variants × 3 sizes × 6 estados), Form fields, Form controls (Checkbox/Radio/Switch com radius 4px exceção), Badge & Pills, Avatar (5 tamanhos + status + stack), Tooltip (4 posições).
  • Components — Card (KPI + Documento), Tabs (underline + pill), Pagination, Loading states (Skeleton + Spinner + Progress), Empty state, Alerts & Banners (inline + banner + toast), Data Table (3 densidades, 8 tipos de célula, estados especiais), Modal & Drawer (4 sizes + confirm + drawer + sheet), Page Header (5 variantes + breadcrumb), Search Input (3 variantes + dropdown), Date picker (single + range + presets), File upload (dropzone + lista + validação + 3 variantes).
  • Navigation — Sidebar (3 estados), Topbar, Notification dropdown, Avatar dropdown.
  • Brand — Logo light & dark, P-cápsula, Branded House Arquitetura, Voice & Microcopy.
  • Templates — App Dashboard (Light + Dark), 404, 500.
  • Documentação — Motion, Voice expandido, Acessibilidade plena, Princípios escritos, Como contribuir, Tokens export.

Decisões fechadas

  • Hover = Focus visualmente em todo o sistema (extensão da regra WCAG: focus precisa ser visível, não diferente de hover).
  • Paleta sem subtons (100/300/500/700/900).
  • Naming técnico em EN, microcopy em PT-BR.
  • Pill reservado a affirmative + primary lg institucional.
  • Radius 4px como exceção para mini-controles (≤ 20px).
  • Opacity removida como foundation; tokens migrados para componentes (backdrop/modal, focus-ring, menu-hover).
  • Auditoria 0.2.1 — consolidação de --neutral-line, --neutral-muted, --brand-neutral-light em --neutral-light; ajustes pontuais de radius fora da escala aplicados nos frames afetados.

Conhecido · registrado para v1.1

  • Capitalização das 8 sub-marcas pendente de decisão de marketing (patrimonium news vs Patrimonium News, radar vs Radar).
  • Loader com identidade Patrimonium — substituir spinner genérico por elemento da marca.
  • Empty states adicionais — sem permissão, sem conexão, error state.
